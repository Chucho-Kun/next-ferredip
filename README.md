# ECommerce Platform NEXT / REACT / TypeScript
ECommerce Platform - Specializing in the sale of tools from different brands and categories, it features a Mercado Pago payment gateway, a PostgreSQL database (railway.com) + Next JS + React + Node - (Full-Stack CRUD Application)
